DRIVE SERVER

Bienvenidos a nuestro tutorial sobre la creación de un servidor de videos desde Google Drive con rclone y Plex.

Paso 1: Descarga y Configura rclone y winfsp. Puedes obtener la última versión de rclone aquí y winfsp. Luego, instala winfsp y configura rclone, entra en la carpeta donde instalaste rclone y en la barra de direcciones utiliza el siguiente comando cmd. Se abre linea de comandos y ahi pega esto: 

rclone config

Paso 2: Monta la carpeta "movies" como la unidad virtual F: con este comando rclone:

rclone mount remote:/movies F:
  • Asegúrate de reemplazar "remote" con el nombre de tu configuración de Google Drive en rclone.
  • Después de ejecutar este comando, tu carpeta "movies" debería ser accesible como la unidad F:.

Después de configurar rclone, instala Plex y añade una nueva biblioteca:

rclone config
Plex Dashboard > Libraries > Add Library > Movies > Specify Name > Add folders (Selecciona la unidad F:)

También cubriremos otros detalles...

Nuestro objetivo es mostrarte cómo disfrutar de las asombrosas posibilidades que ofrece esta configuración...

Esperamos que disfrutes aprendiendo sobre las increíbles posibilidades que te brinda este servidor.

Tutorial: Transmisión de Video en Vivo con Nginx

Bienvenido al tutorial sobre cómo configurar y usar un servidor de video en vivo con Nginx. En este tutorial, aprenderás cómo configurar un servidor de video en vivo usando Nginx y cómo crear una página web para interactuar con el video transmitido.

Paso 1: Configuración del Servidor Nginx

Primero, necesitas tener instalado Nginx en tu sistema. Puedes instalarlo utilizando el gestor de paquetes de tu sistema operativo.

# Ubuntu
sudo apt update
sudo apt install nginx

Luego, crea un archivo de configuración para Nginx. Aquí tienes un ejemplo básico:

# Archivo de configuración nginx.conf
worker_processes  2;

events {
    worker_connections  8192;
}

http {
    server {
        listen 8080;

        location /live {
            # Configuración HLS
            ...
        }     
    }
}

rtmp {
    server {
        listen 1935;
        ...

        application live {
            live on;
            ...

            # Configuración HLS
            hls on;
            hls_path /nginx/temp/live;
        }
    }
}

Paso 2: Creación de la Página HTML

Ahora, crearemos una página HTML para interactuar con el video transmitido. Aquí tienes un ejemplo:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Transmisión de Video en Vivo</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="container">
        <h1>Transmisión de Video en Vivo</h1>
        <video id="videoPlayer" controls></video>
        <button id="startButton">Iniciar</button>
        <button id="stopButton">Detener</button>
    </div>

    <script src="script.js"></script>
</body>
</html>

Paso 3: Estilo y Funcionalidad Interactiva

Finalmente, agregaremos estilo y funcionalidad interactiva a nuestra página HTML mediante CSS y JavaScript:

/* styles.css */
body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 0;
    background-color: #f4f4f4;
}

.container {
    max-width: 800px;
    margin: 50px auto;
    text-align: center;
}

h1 {
    color: #333;
}

video {
    width: 100%;
    margin-bottom: 20px;
}

button {
    padding: 10px 20px;
    font-size: 16px;
    cursor: pointer;
    background-color: #007bff;
    color: #fff;
    border: none;
    border-radius: 4px;
}

button:hover {
    background-color: #0056b3;
}

/* script.js */
const videoPlayer = document.getElementById('videoPlayer');
const startButton = document.getElementById('startButton');
const stopButton = document.getElementById('stopButton');

const videoSource = 'http://192.168.0.183:8080/live/stream';

startButton.addEventListener('click', () => {
    videoPlayer.src = videoSource;
    videoPlayer.play();
});

stopButton.addEventListener('click', () => {
    videoPlayer.pause();
    videoPlayer.src = '';
});

REMOTE SERVER

Lleva tu servidor local a las nubes con Pyngrok

Paso 1: Descarga y Configura rclone. Puedes obtener la última versión de rclone aquí. Luego, configura rclone, entra en la carpeta donde instalaste rclone y en la barra de direcciones utiliza el siguiente comando cmd. Se abre linea de comandos y ahi pega esto: 

rclone config

Paso 2: Sirve la carpeta "movies" como contenido online con este comando rclone:

      rclone serve http remote:/movies
  • Asegúrate de reemplazar "remote" con el nombre de tu configuración de Google Drive en rclone.
  • Después de ejecutar este comando, tu carpeta "movies" debería ser accesible en localhost:8080.

Paso 3: Haz que tu servidor sea accesible en línea con Pyngrok. Descarga e instala Pyngrok desde aquí. Luego, ejecuta los siguientes comandos abriendo otra terminal cmd:

      ngrok config add-authtoken "tu token"
      pyngrok http 8080

Listo, copia la url generada y abrela con cualquier browser y veras todo el contenido de "movies"
